# Web Tasarım Proje Ödevi

Sakarya Üniversitesi Bilgisayar Mühendisliği Bölümü web tasarım dersi kapsamında geliştirilmiş kişisel web sitesi projesidir.

## Proje Yapısı

```
├── index.html          # Ana sayfa / Hakkımda
├── sehrim.html         # Şehrim - Trabzon
├── takımım.html        # Trabzonspor
├── ilgialanlarım.html  # İlgi Alanlarım
├── ozgecmis.html       # Özgeçmiş / CV
├── iletisim.html       # İletişim formu
├── login.html          # Giriş sayfası
├── css/
│   └── style.css       # Stil dosyası
├── js/
│   └── script.js       # JavaScript dosyası
├── resimler/           # Görsel içerikler
├── login.php           # Giriş backend
└── iletisimdonus.php   # İletişim formu backend
```

## Teknolojiler

- **HTML5** – Semantik işaretleme
- **CSS3** – Responsive tasarım, Grid/Flexbox
- **Bootstrap 5** – UI bileşenleri
- **JavaScript** – Form validasyonu, interaktif özellikler
- **PHP** – Giriş ve iletişim formu işlemleri
- **Font Awesome** – İkon seti

## Özellikler

- Responsive tasarım (mobil uyumlu)
- Bootstrap 5 ile modern arayüz
- Trabzonspor ve Trabzon içerikleri
- Özgeçmiş / CV sayfası
- İletişim formu (PHP ile işleme)
- Giriş sayfası (PHP ile kimlik doğrulama)

## Kurulum

1. Projeyi klonlayın: `git clone https://github.com/m-salihtysz/Web-Tasarim-Proje-Odevi.git`
2. PHP kullanan sayfalar için yerel sunucu gerekir (ör. XAMPP, WAMP veya `php -S localhost:8000`)
3. Tarayıcıda `index.html` veya sunucu adresini açın
